# Keyboard-Shortcuts
C program for auto-writing with custom keyboard shortcuts in Linux systems

## Dependencies
The only dependencie needed is libxtst-dev which you can install manually or using make file:
```
make install
```
## Compiling
```
make
```
## Running
sudo ./snippets
